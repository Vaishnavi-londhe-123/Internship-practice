# Employee REST API using Flask

## Project Description
This project is a REST API developed using Python Flask and SQLAlchemy. It performs CRUD operations on employee records stored in a MySQL database.

## Technologies Used
- Python
- Flask
- Flask SQLAlchemy
- MySQL
- Postman

## Database
Database Script: empmgt.sql

## API Endpoints

### Add Employee
POST /employee

### Get All Employees
GET /employees

### Get Employee By ID
GET /employee/<id>

### Update Employee
PUT /employee/<id>

### Delete Employee
DELETE /employee/<id>
